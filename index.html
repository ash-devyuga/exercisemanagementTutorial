1. Creating New Cases and Schedules

The mobile app allows you to create new patient cases and schedules directly from your device.

Steps to Create a New Case:
  1. 1 Navigate to the patient profile
  2. 2 Tap "Add Case" or the "+" button
  3. 3 Enter the Case Name (e.g., "case demo 123")
  4. 4 Enter the Schedule Name (e.g., "schedule 123")
  5. 5 Set Schedule Duration:
    • Select Start Date (e.g., Nov 10, 2025)
    • Select End Date (e.g., Feb 10, 2026)
  6. 6 Tap "Create" to save the new case and schedule
